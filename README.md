# Capsule Dashboard

Mini projet de démonstration réalisé avec **React**, **Next.js (App Router)**, **TypeScript**, **TailwindCSS** et **Shadcn UI**.

## ✨ Objectif

Montrer un exemple de tableau de bord B2B moderne, dans le cadre d’une mission frontend pour une refonte UI (type Comet ou client direct).

## 🧰 Stack technique

- React 18 / Next.js 14
- TypeScript
- TailwindCSS
- Shadcn UI
- Déploiement : [Vercel](https://vercel.com)
